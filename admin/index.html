上传内容（按文件夹自动分类）

想加哪类内容，就点对应分类的「上传」按钮，把 HTML 拖进去即可——系统按文件夹自动归类，无需填任何标签。

📤 选择分类上传

点击下方任一分类的「上传到这里」，会直接打开该文件夹的 GitHub 上传页面：拖入 HTML → 点绿色按钮 Commit changes → 约 1 分钟后自动上线。

正在加载分类…

🔁 上传后会发生什么

  1. 你上传文件到某个分类文件夹（例如「单词」文件夹）。
  2. 系统自动识别该文件夹对应的板块与科目，重建内容列表（约 1 分钟）。
  3. 学生刷新前台即可在对应分类看到新内容，你无需做任何其他操作。

删除内容：进入对应分类文件夹，点开文件 → 右上角垃圾桶图标删除即可，列表会自动更新。

✨ 可选：给文件美化标题 / 简介

分类已由文件夹决定，这一步不是必须的。若想自定义卡片上显示的标题、时长或简介，可用此工具处理后再上传。

📄
点击选择 或把 HTML 文件拖到这里

下载后，把这个新文件上传到对应分类文件夹即可（见最上方「选择分类上传」）。

📊 进阶：让成绩自动记录到「我的成绩」

内容自带批改即可正常使用。若希望分数自动同步，在算出分数后加一行：

window.parent.postMessage( { type: "yysd:score", score: 32, total: 40, band: 7 }, "*");

示范文件（语法、单词、阅读、入学模考）均已内置该写法，可参考各分类文件夹。

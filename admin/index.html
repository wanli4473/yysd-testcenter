上传模考 / 学习册

无需编程。准备好 HTML 内容，选择板块和科目，按下面三步操作即可上线。

🗂️ 三大板块结构

上传时为内容选定「板块 + 科目」，它就会自动出现在前台对应分类下。

  • 🎯 模考区：雅思模考 · PTE模考 · 托福模考
  • 📚 学习区：语法学习 · 单词学习
  • ✏️ 练习区：雅思练习题 · PTE练习题 · 托福练习题

📥 上传流程（全程网页点击，无需命令行）

  1. 整理内容（推荐）：把 HTML 文件拖到下方工具区，选好板块/科目并填写标题，点击「生成并下载」。下载到的新文件已带好标签。
  2. 上传到 GitHub：打开仓库的 library 文件夹 → 点 Add file → Upload files → 拖入 HTML → 点 Commit changes。
  3. 自动上线：上传后系统会自动重建内容列表（约 1 分钟），刷新前台即可看到，无需任何其他操作。

删除内容同理：在 library 文件夹删除对应 HTML 文件，列表会自动更新。

① 整理内容（打标签工具）

在你的浏览器本地运行，不会上传任何文件。处理完直接下载到本机。

点击选择 或把 HTML 文件拖到这里

下载后，把这个新文件上传到仓库的 library 文件夹即可（见上方第 2 步）。

② 进阶：手动给内容加标签

直接编辑 HTML 时，把下面这段粘到 <head> 里改成你的内容即可。系统会优先读取这些标签。

<meta name="exam:title" content="雅思全真模考（第四卷）"> <meta name="exam:zone" content="mock"> <!-- mock 模考 / study 学习 / practice 练习 --> <meta name="exam:subject" content="ielts"> <!-- 模考/练习: ielts pte toefl ｜ 学习: grammar vocab --> <meta name="exam:duration" content="0"> <!-- 分钟，0=不限时 --> <meta name="exam:description" content="3 篇文章共 40 题。">

不加也没关系——系统会根据文件名和网页标题自动猜测板块与科目。

③ 进阶：让成绩自动记录到「我的成绩」

你的内容自带批改即可正常使用。若希望分数自动同步，在算出分数后加一行：

window.parent.postMessage( { type: "yysd:score", score: 32, total: 40, band: 7 }, "*");

示范文件（语法、单词、阅读练习）均已内置该写法，可参考 library/ 目录。

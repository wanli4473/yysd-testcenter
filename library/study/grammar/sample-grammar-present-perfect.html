现在完成时 Present Perfect

学习区 · 语法精讲 · 约 10 分钟

一、基本构成

主语 + have / has + 动词的过去分词 (past participle)
  • I / you / we / they → have done
  • he / she / it → has done

e.g. She has finished her homework.

二、三大核心用法

  • 过去发生、对现在有影响：I have lost my keys.（现在还没找到）
  • 从过去持续到现在：常与 for / since 连用。We have lived here for ten years.
  • 人生经历、刚刚完成：常与 ever / never / already / just / yet 连用。Have you ever been to London?

三、常见信号词

just / already / yet / ever / never / since / for / recently / so far

随堂小测 (6 题)

选择或填空，完成后点击「提交批改」。

1. He __________ in Beijing since 2015.

答案：has lived —— since 引导，从过去持续到现在。

2. We __________ all the exercises already.

答案：have finished —— already 是现在完成时信号词。

3. 填入 see 的正确形式：Have you ever __________ a panda?

答案：seen（see 的过去分词）

4. She isn't here; she __________ to the bank.

答案：has gone —— 强调"去了还没回"，对现在有影响。

5. 填 for 或 since：I have known him __________ five years.

答案：for —— 后接"一段时间"用 for。

6. 填否定缩写：They __________ done it yet.（have + not）

答案：haven't（have not 亦可）
